Raonament:
Per a rotar un punt (x, y) al voltant de l'origen per un angle de 45 graus en sentit antihorari, usem la següent fórmula:
x' = x * cos(45) - y * sin(45)
y' = x * sin(45) + y * cos(45)
Com que cos(45) = sin(45) = √2 / 2, tenim:
x' = (x - y) * √2 / 2
y' = (x + y) * √2 / 2
Amb el punt (7, 2):
x' = (7 - 2) * √2 / 2 = 5√2 / 2
y' = (7 + 2) * √2 / 2 = 9√2 / 2
Per tant, les noves coordenades són (5√2 / 2, 9√2 / 2).
