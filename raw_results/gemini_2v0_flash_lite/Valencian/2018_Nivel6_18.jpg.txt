1. **Identificar les relacions de les arrels amb els coeficients**: Per a una equació quadràtica de la forma ax² + bx + c = 0, la suma de les arrels (a + b) és -b/a, i el producte de les arrels (ab) és c/a. En aquest cas, l'equació és x² + x - 7 = 0, on a = 1, b = 1 i c = -7. Per tant, a + b = -1/1 = -1 i ab = -7/1 = -7.

2. **Reescriure l'expressió per facilitar la substitució**:
Volem trobar el valor de 3a² + 4b² + 2a + 3b + 1.
Podem reescriure l'expressió usant l'equació original per expressar a² i b².
Des de x² + x - 7 = 0, sabem que x² = 7 - x. Per tant, a² = 7 - a i b² = 7 - b.
Substituïm a² i b² a l'expressió original:
3(7 - a) + 4(7 - b) + 2a + 3b + 1
= 21 - 3a + 28 - 4b + 2a + 3b + 1
= 50 - a - b
= 50 - (a + b)
= 50 - (-1)
= 50 + 1
= 51
