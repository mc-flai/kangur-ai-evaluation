Raonament:
1. Factorem l'equació ac + ad + bc + bd = 68. Podem agrupar els termes per factoritzar:
   (ac + ad) + (bc + bd) = 68
   a(c + d) + b(c + d) = 68
   (a + b)(c + d) = 68
2. Sabem que c + d = 4. Substituïm a l'equació:
   (a + b) * 4 = 68
   a + b = 68 / 4
   a + b = 17
3. Volem trobar el valor de a + b + c + d. Sabem que a + b = 17 i c + d = 4.
   a + b + c + d = (a + b) + (c + d) = 17 + 4 = 21.
