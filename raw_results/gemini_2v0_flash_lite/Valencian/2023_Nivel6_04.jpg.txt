Raonament: Hem de comprovar quins parells de punts compleixen l'equació y = bx + 1.
*   **A**: (0,0): 0 = b\*0 + 1 => 0 = 1 (Fals).  (1/b, 0): 0 = b\*(1/b) + 1 => 0 = 1+1 => 0 = 2 (Fals).

*   **C**: (0,1): 1 = b\*0 + 1 => 1 = 1 (Veritat). (b,0): 0 = b\*b + 1 => -1 = b² (Fals per nombres reals).
