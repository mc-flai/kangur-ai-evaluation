Raonament: L'equació donada és x³ + y³ = x²y². Per analitzar els quadrants, podríem considerar alguns casos. Si x = 0, llavors y³ = 0, per tant y = 0. Això ens dóna el punt (0,0). Si y = 0, llavors x³ = 0, per tant x = 0. Això també ens dóna el punt (0,0). Per tant, l'origen (0,0) està a la gràfica. Per tant, la gràfica passa per tots els quadrants.
