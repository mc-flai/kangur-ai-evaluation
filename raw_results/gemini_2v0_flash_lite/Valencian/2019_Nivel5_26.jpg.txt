Raonament:
1. **Condicions dels nombres:** Els nombres *a*, *b*, i *c* són de tres xifres, amb la primera i l'última xifra iguals. Per tant, *a* és de la forma X_Y_X, on X i Y són dígits.
2. **Relacions entre els nombres:**
   *b* = 2*a* + 1
   *c* = 2*b* + 1
3. **Substitució:** Substituïm el valor de *b* a la segona equació:
   *c* = 2*(2*a* + 1) + 1 = 4*a* + 2 + 1 = 4*a* + 3
4. **Interval de *a*:** Els nombres són entre 100 i 999. Atès que *a* té la forma X_Y_X, la primera xifra pot ser qualsevol entre 1 i 9. Per tant, *a* pertany a l'interval [101, 999].
5. **Interval de *b*:** *b* = 2*a* + 1. El valor de *b* també ha de complir les condicions del problema (format X_Y_X).
6. **Interval de *c*:** *c* = 2*b* + 1, i *c* té la forma X_Y_X.
7. **Càlculs i anàlisi:**
   * Si a = 101, b = 2*101 + 1 = 203. c = 2*203 + 1 = 407. El número a compleix les condicions.
   * Si a = 202, b = 2*202 + 1 = 405. c = 2*405 + 1 = 811. El número a compleix les condicions.
   * Si a = 303, b = 2*303 + 1 = 607. c = 2*607 + 1 = 1215. El número c no compleix les condicions (no és un nombre de tres xifres).
   * Si a augmenta, *c* augmenta més ràpidament i deixarà de ser un nombre de tres xifres.
   * Hem trobat dos valors possibles per *a*: 101 i 202.

Resposta: C) 2
