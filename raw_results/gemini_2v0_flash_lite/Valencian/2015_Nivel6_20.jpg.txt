Raonament: The problem asks for the number of different numerical values that can be obtained by inserting parentheses in the expression A - B x C + D x E.  Let's calculate the value of the expression without parentheses: 2015 - 2016 * 1 + 2 * 3 = 2015 - 2016 + 6 = 5.
Now consider possible parentheses:
1.  (A - B) x C + D x E = (2015 - 2016) * 1 + 2 * 3 = -1 + 6 = 5
2.  A - (B x C) + D x E = 2015 - (2016 * 1) + 2 * 3 = 2015 - 2016 + 6 = 5
3.  A - B x (C + D) x E = 2015 - 2016 * (1 + 2) * 3 = 2015 - 2016 * 9 = 2015 - 18144 = -16129
4.  A - B x C + (D x E) = 2015 - 2016 * 1 + (2 * 3) = 2015 - 2016 + 6 = 5
5.  (A - B x C) + D x E = (2015 - 2016) + 6 = -1 + 6 = 5
6.  A - (B x C + D) x E = 2015 - (2016 + 2) * 3 = 2015 - 2018 * 3 = 2015 - 6054 = -4039
7.  A - (B x C + D x E) = 2015 - (2016 + 6) = 2015 - 2022 = -7
8.  (A - B x C + D) x E = (2015-2016 + 2) * 3 = (1) * 3 = 3
9.  (A - B) x (C + D x E) = (2015 - 2016) * (1 + 6) = -1 * 7 = -7
So, different values are 5, -16129, -4039, 3, -7.
Thus there are 5 different values.
