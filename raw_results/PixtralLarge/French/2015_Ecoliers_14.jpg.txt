La question 14 demande de trouver le dessin vu de dessus d'un parapluie anglais dont le mot « KANGAROO » est écrit dessus. Pour résoudre ce problème, nous devons comprendre comment les lettres s'inscrivent sur le parapluie quand on regarde le parapluie par le dessus.

Les lettres du mot « KANGAROO » doivent être lues dans le sens des aiguilles d'une montre.

Voici la disposition des lettres de « KANGAROO »:

- K
- A
- N
- G
- A
- R
- O
- O

En les inscrivant dans le sens des aiguilles d'une montre sur le parapluie, la séquence doit être : K, A, N, G, A, R, O, O.

En regardant les options données :

- Option A) : N, A, G, R, O, K
- Option B) : R, O, O, K, A, N, G
- Option C) : O, N, R, K, G, A, A
- Option D) : G, R, A, N, O, K, A
- Option E) : G, N, A, R, A, K, O, O

Seule la séquence de l'option E) suit la disposition correcte des lettres « KANGAROO » dans le sens des aiguilles d'une montre.

Raisonnement : En plaçant les lettres « KANGAROO » dans le sens des aiguilles d'une montre sur le parapluie, la séquence vue de dessus est G, N, A, R, A, K, O, O, ce qui correspond à l'option E).

Réponse : E).
