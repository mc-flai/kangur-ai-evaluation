Per determinar quantes diners porten entre Bernat i David, és necessari establir els diners que porta cada persona i després calcular la diferència.

Anem a anomenar D a la quantitat de diners que porta cada persona:
- Alba i Bernat porten entre tots dos 5 €.
- Cris i David porten entre tots dos 6 €.
- Alba i Cris porten entre tots dos 7 €.

Comencem a assignar valors a les quantitats de diners:
1. Alba = A
2. Bernat = B
3. Cris = C
4. David = D

De les dades tenim:
A + B = 5 €
C + D = 6 €
A + C = 7 €

Sumem les dues primeres equacions:
(A + B) + (C + D) = 5 + 6
A + B + C + D = 11 €

Ara restem la tercera equació:
A + B + C + D - (A + C) = 11 - 7
B + D = 4 €

Per tant, Bernat i David porten entre tots dos 4 €.

Raonament: Hem utilitzat les dades del problema per establir un sistema d'equacions i després les hem resolt per trobar la quantitat que porten Bernat i David.
