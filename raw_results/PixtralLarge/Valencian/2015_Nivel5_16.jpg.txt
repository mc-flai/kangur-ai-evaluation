**Raonament**:
Per determinar si el punt \( P(2, -1) \) pertany a la recta representada per l'equació \( ax + by = c \), hem de substituir els valors de \( x = 2 \) i \( y = -1 \) en l'equació i comprovar si la igualtat es compleix.

Substituint els valors en l'equació \( ax + by = c \) obtenim:
\[ a(2) + b(-1) = c \]
\[ 2a - b = c \]

Ara, revisem cada una de les opcions per veure quina satisfà aquesta igualtat:

A) \( a = -1, b = 1, c = -3 \)
\[ 2(-1) - 1 = -3 \]
\[ -2 - 1 = -3 \]
\[ -3 = -3 \]
Aquesta opció és correcta.

B) \( a = 1, b = 0, c = 2 \)
\[ 2(1) - 0 = 2 \]
\[ 2 = 2 \]
Aquesta opció és correcta.

C) \( a = 0, b = 3, c = -3 \)
\[ 2(0) - 3 = -3 \]
\[ 0 - 3 = -3 \]
\[ -3 = -3 \]
Aquesta opció és correcta.

D) \( a = 10, b = 15, c = 5 \)
\[ 2(10) - 15 = 5 \]
\[ 20 - 15 = 5 \]
\[ 5 = 5 \]
Aquesta opció és correcta.

E) \( a = 1, b = 2, c = 1 \)
\[ 2(1) - 2 = 1 \]
\[ 2 - 2 = 0 \]
\[ 0 \neq 1 \]
Aquesta opció no és correcta.

Totes les opcions excepte E són correctes, ja que satisfan l'equació \( 2a - b = c \). Per tant, hi ha més d'una opció vàlida, el que significa que no hi ha una única elecció possible.

**Resposta**: No hi ha una única resposta correcta.
