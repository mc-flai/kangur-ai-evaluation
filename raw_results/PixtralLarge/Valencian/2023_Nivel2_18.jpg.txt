Per resoldre aquesta pregunta, primer necessitem entendre les condicions donades:

1. La suma dels tres primers nombres és 100.
2. La suma dels tres nombres que estan al mig és 200.
3. La suma dels tres últims nombres és 300.

Suposem que els cinc nombres de la llista són: a, b, c, d, e.

Donades les condicions:
a + b + c = 100
b + c + d = 200
c + d + e = 300

(a + b + c) + (b + c + d) + (c + d + e) = 100 + 200 + 300
a + 2b + 3c + 2d + e = 600
