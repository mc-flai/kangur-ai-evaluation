**Raonament**:
Eva està fent un model d’un brick d’un litre de llet. Si les cares del model són la meitat de les cares de l'original, això significa que les dimensions del model són la meitat de les dimensions de l'original en cada una de les tres dimensions (longitud, amplada i alçada).

Per il·lustrar, si les dimensions de l'original són \( a \times b \times c \), llavors les dimensions del model seran \( \frac{a}{2} \times \frac{b}{2} \times \frac{c}{2} \).

El volum de l'original és \( a \times b \times c \).

El volum del model serà:
\[ \left(\frac{a}{2}\right) \times \left(\frac{b}{2}\right) \times \left(\frac{c}{2}\right) = \frac{a}{2} \times \frac{b}{2} \times \frac{c}{2} = \frac{a \times b \times c}{8} \]

Això significa que el volum del model és \(\frac{1}{8}\) del volum de l'original.

Donat que l'original és d'1 litre, el volum del model és:
\[ \frac{1}{8} \text{ l} = 0.125 \text{ l} \]

**Resposta**: D) 0,125 l
