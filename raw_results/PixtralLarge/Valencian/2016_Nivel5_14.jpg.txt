Raonament: Per trobar el valor de \(a + b + c + d\), utilitzarem les dades proporcionades en l'enunciat.

Primer, ens donen \(ac + ad + bc + bd = 68\) i \(c + d = 4\).

Podem factoritzar l'expressió \(ac + ad + bc + bd\) com:
\[ ac + ad + bc + bd = (a + b)(c + d) \]

Sabem que \(c + d = 4\). Sustituïm aquest valor en l'expressió factoritzada:
\[ (a + b)(c + d) = 68 \]
\[ (a + b)(4) = 68 \]

Ara dividim la dada per 4 per trobar \(a + b\):
\[ a + b = \frac{68}{4} \]
\[ a + b = 17 \]

Per trobar el valor total de \(a + b + c + d\), sumem \(a + b\) amb \(c + d\):
\[ a + b + c + d = 17 + 4 \]
\[ a + b + c + d = 21 \]

Resposta: C)
