
La question 22 demande de combiner les équations données pour trouver la valeur de x + y.

Équations données :
1. |x| + x + y = 5
2. x + |y| - y = 10

Pour résoudre ces équations, nous devons d'abord déterminer les valeurs de x et y en tenant compte des valeurs absolues.

Raisonnement :
1. Pour la première équation, |x| + x + y = 5 :
   - Si x ≥ 0, alors |x| = x.
   - Si x < 0, alors |x| = -x.
   - Nous devons déterminer si x est positif ou négatif pour résoudre cette équation.

2. Pour la deuxième équation, x + |y| - y = 10 :
   - Si y ≥ 0, alors |y| = y.
   - Si y < 0, alors |y| = -y.
   - Nous devons déterminer si y est positif ou négatif pour résoudre cette équation.

Combinons les deux équations pour trouver x et y.

Équation combinée :
|x| + x + y + x + |y| - y = 5 + 10
2x + |y| - y = 15

Pour résoudre cette équation, nous devons déterminer les valeurs de x et y qui satisfont les deux équations.

- Si x ≥ 0, alors |x| = x, donc |x| + x = 2x.
- Si y ≥ 0, alors |y| = y.
- Si y < 0, alors |y| = -y.

Essayons différentes valeurs pour x et y pour voir lesquelles satisfont les deux équations.

Pour x ≥ 0 :
|x| + x + y = 5
2x + y = 5

Pour y ≥ 0 :
x + |y| - y = 10
x + y - y = 10
x = 10

Substituons x = 10 dans l'équation 2x + y = 5 :
2(10) + y = 5
20 + y = 5
y = 5 - 20
y = -15

Donc, x = 10 et y = -15 satisfont les deux équations.

Réponse : C
