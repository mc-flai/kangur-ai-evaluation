
Raisonnement :
1. Le problème indique que le nombre de réponses justes est le double du nombre de réponses fausses.
2. Soit x le nombre de réponses fausses. Par conséquent, le nombre de réponses justes est 2x.
3. Le total des réponses justes et fausses est de 60.
4. Nous pouvons écrire l'équation suivante :
   x (réponses fausses) + 2x (réponses justes) = 60
5. Simplifier l'équation :
   3x = 60
6. Résoudre pour x :
   x = 60 / 3
   x = 20
7. Donc, le nombre de réponses fausses est 20 et le nombre de réponses justes est 2 * 20 = 40.
8. Nous devons trouver une combinaison de questions a-t-elle toutes répondu juste.
9. Les options sont :
   A) 20
   B) 30
   C) 32
   D) 36
   E) 40
10. La combinaison de questions a-t-elle toutes répondu juste est le nombre de réponses justes, qui est 40.

Réponse : E)
