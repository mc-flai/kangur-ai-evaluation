Para resolver la pregunta 21, necesitamos determinar cuántos números positivos se pueden borrar de un número original para obtener un número igual a 1/14 del número original.

1. Primero, identifiquemos el número original. El problema no proporciona el número original, pero podemos suponer que es un número positivo.

2. El número original se divide por 14 para obtener 1/14 del número original. Por lo tanto, el número original es 14 veces el número que obtenemos después de borrar los números positivos.

3. Ahora, determinemos qué números positivos se pueden borrar de un número para obtener un número que sea 1/14 del número original. Esto implica que el número después de borrar debe ser divisible por 14.

4. Consideremos las opciones dadas:
   - A) 0
   - B) 1
   - C) 2
   - D) 3
   - E) 4

5. Evaluemos cada opción:
   - A) 0: Borrar 0 números no cambia el número original.
   - B) 1: Borrar 1 no cambia significativamente el número original.
   - C) 2: Borrar 2 números no cambia significativamente el número original.
   - D) 3: Borrar 3 números no cambia significativamente el número original.
   - E) 4: Borrar 4 números no cambia significativamente el número original.

6. Ninguna de las opciones parece ser una cantidad significativa de números positivos que se pueden borrar para obtener 1/14 del número original.

Razonamiento: Ninguna de las opciones proporcionadas parece ser una cantidad significativa de números positivos que se pueden borrar para obtener 1/14 del número original.

Respuesta: A)
