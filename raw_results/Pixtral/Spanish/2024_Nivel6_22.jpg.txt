Para determinar cuál de las opciones es correcta, primero debemos entender las ecuaciones dadas y las afirmaciones que se hacen sobre ellas.

Las ecuaciones son:
1. \( ax^2 + bx + c = 0 \)
2. \( bx^2 + ax + c = 0 \)

Las afirmaciones son:
- La solución común debe ser 0.
- La ecuación cuadrática \( ax^2 + bx + c = 0 \) tiene exactamente una solución real.
- \( a > 0 \)
- \( b < 0 \)
- \( a + b + c = 0 \)

Razonamiento:
1. Si la solución común debe ser 0, esto implicaría que las raíces de ambas ecuaciones son las mismas. Sin embargo, esto no necesariamente es cierto para todas las ecuaciones cuadráticas.
2. La afirmación de que la ecuación cuadrática \( ax^2 + bx + c = 0 \) tiene exactamente una solución real implica que el discriminante de esta ecuación es cero. El discriminante se calcula como \( b^2 - 4ac \). Para que haya exactamente una solución real, \( b^2 - 4ac = 0 \).
3. La afirmación \( a > 0 \) no nos da información directa sobre las raíces o el discriminante.
4. La afirmación \( b < 0 \) no nos da información directa sobre las raíces o el discriminante.
5. La afirmación \( a + b + c = 0 \) no nos da información directa sobre las raíces o el discriminante.

Dado que la afirmación clave es que la ecuación cuadrática \( ax^2 + bx + c = 0 \) tiene exactamente una solución real, podemos concluir que la afirmación correcta debe ser la que se relaciona con el discriminante.

Respuesta: B)
