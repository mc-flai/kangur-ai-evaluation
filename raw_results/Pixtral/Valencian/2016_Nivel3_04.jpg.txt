**Pregunta**: Quantos números enteros hay de tres cifras cuya suma de sus cifras es igual a 3?

**Opcions**:
A) 3
B) 4
C) 5
D) 6
E) 8

**Razonamiento**:
Para resolver este problema, necesitamos encontrar números de tres cifras (de la forma abc) cuya suma de cifras sea igual a 3.

1. Identificamos los números de tres cifras posibles.
2. Calculamos la suma de sus cifras.
3. Verificamos si la suma es igual a 3.

Veamos algunos ejemplos:
- 100: 1 + 0 + 0 = 1 (no cumple)
- 200: 2 + 0 + 0 = 2 (no cumple)
- 300: 3 + 0 + 0 = 3 (cumple)
- 110: 1 + 1 + 0 = 2 (no cumple)
- 220: 2 + 2 + 0 = 4 (no cumple)
- 330: 3 + 3 + 0 = 6 (no cumple)
- 121: 1 + 2 + 1 = 4 (no cumple)
- 212: 2 + 1 + 2 = 5 (no cumple)
- 321: 3 + 2 + 1 = 6 (no cumple)
- 102: 1 + 0 + 2 = 3 (cumple)
- 201: 2 + 0 + 1 = 3 (cumple)
- 101: 1 + 0 + 1 = 2 (no cumple)
- 210: 2 + 1 + 0 = 3 (cumple)

De esta manera, los números de tres cifras cuya suma de cifras es igual a 3 son: 300, 102, 201, 210. Hay 4 números que cumplen con esta condición.
