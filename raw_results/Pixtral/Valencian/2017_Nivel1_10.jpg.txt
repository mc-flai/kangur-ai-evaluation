La pregunta 10 és: "Quina figura té l'àrea més gran?"

Les opcions són:
A) Una figura amb una forma irregular.
B) Una estrella de cinc puntes.
C) Una estrella de sis puntes.
D) Una figura amb una forma similar a una nube.
E) Una estrella de set puntes.

**Raonament:**
1. Per determinar quina figura té l'àrea més gran, primer cal calcular l'àrea de cada figura.
2. Les estrelles de puntes múltiples (com les estrelles de cinc puntes, sis puntes i set puntes) tenen àrees que poden ser més grans que les figures més simples.
3. La figura A i la figura D tenen formes més simples i més petites.
4. La figura B (estrella de cinc puntes) i la figura C (estrella de sis puntes) tenen àrees més grans que les figures A i D.
5. La figura E (estrella de set puntes) té més puntes i, per tant, una àrea més gran que les estrelles de cinc puntes i sis puntes.

Per tant, la figura E té l'àrea més gran.

**Resposta:** E)
