La pregunta 9 pregunta cuál es el cuadrilátero de la figura de la derecha que tiene la mayor área.

Para determinar cuál de los cuadriláteros tiene la mayor área, debemos calcular el área de cada uno. El área de un cuadrilátero puede determinarse utilizando la fórmula:

\[ \text{Área} = \frac{1}{2} \times \text{base} \times \text{altura} \]

Vamos a calcular el área de cada uno de los cuadriláteros:

1. Cuadrilátero 1:
   - Base = 1
   - Altura = 1
   - Área = \(\frac{1}{2} \times 1 \times 1 = \frac{1}{2}\)

2. Cuadrilátero 2:
   - Base = 2
   - Altura = 2
   - Área = \(\frac{1}{2} \times 2 \times 2 = 2\)

3. Cuadrilátero 3:
   - Base = 3
   - Altura = 3
   - Área = \(\frac{1}{2} \times 3 \times 3 = \frac{9}{2}\)

4. Cuadrilátero 4:
   - Base = 4
   - Altura = 4
   - Área = \(\frac{1}{2} \times 4 \times 4 = 8\)

5. Cuadrilátero 5:
   - Base = 5
   - Altura = 5
   - Área = \(\frac{1}{2} \times 5 \times 5 = \frac{25}{2}\)

Comparando las áreas calculadas:

- Cuadrilátero 1: \(\frac{1}{2}\)
- Cuadrilátero 2: 2
- Cuadrilátero 3: \(\frac{9}{2}\)
- Cuadrilátero 4: 8
- Cuadrilátero 5: \(\frac{25}{2}\)

El cuadrilátero con la mayor área es el Cuadrilátero 5.
