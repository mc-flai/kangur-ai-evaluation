La pregunta 4 en la imatge es tradueix com a: "4. Tirem tres dels estandards simultàniament i somem els punts obtinguts. Quantes sumes diferents podem obtenir?"

Per resoldre aquesta pregunta, primer hem de comprendre el concepte de combinació. Estem treballant amb tres estandards i somant els punts obtinguts. Cada estandard té un nombre de punts associat i volem trobar totes les sumes possibles d'aquests tres nombres.

Suposem que els estandards són A, B i C, i els seus punts associats són a, b i c, respectivament. Les sumes diferents que podem obtenir són:

1. a + b + c
2. a + b + c
3. a + b + c
4. a + b + c

Però, si els estandards tenen valors diferents, les sumes també seran diferents. Si els estandards tenen els mateixos valors, les sumes seran les mateixes.

Per exemple, si els estandards tenen els valors 1, 2 i 3, les sumes possibles són:

1. 1 + 2 + 3 = 6
2. 1 + 3 + 2 = 6
3. 2 + 1 + 3 = 6
4. 2 + 3 + 1 = 6
5. 3 + 1 + 2 = 6

Però, si els estandards tenen valors diferents, les sumes seran diferents. Per exemple, si els estandards tenen els valors 1, 2 i 4, les sumes possibles són:

1. 1 + 2 + 4 = 7
2. 1 + 4 + 2 = 7
3. 2 + 1 + 4 = 7
4. 2 + 4 + 1 = 7
5. 4 + 1 + 2 = 7

Per tant, les sumes diferents que podem obtenir són 5.

Raonament: Combinem els estandards i somem els punts associats per trobar totes les sumes possibles. Si els estandards tenen valors diferents, les sumes seran diferents.
