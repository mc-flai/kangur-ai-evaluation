Razonamiento: Sea *x* la longitud del lado del triángulo equilátero ABC. Entonces el perímetro de ABC es 3*x*. Los triángulos AEF, DFE y CDF son isósceles e idénticos. Sea *y* la longitud de los lados iguales de estos triángulos. Entonces AF = FC = AE = ED = *y*.
El perímetro de AEF = AE + EF + FA = 2*y* + EF. Similarmente, el perímetro de DFE = DF + FE + ED = 2*y* + EF y el perímetro de CDF = CD + DF + FC = 2*y* + DF.
Como los cuatro triángulos tienen el mismo perímetro, podemos decir que DF = EF. Además, como AEF, DFE y CDF son idénticos, los lados isósceles tienen la misma longitud *y*. Entonces AE = AF = FC = CD = DE = *y*.
El perímetro de ABC = perímetro de AEF.
Por lo tanto, 3*x* = 2*y* + EF.
El perímetro del pentágono ABCDE = AB + BC + CD + DE + EA = *x* + *x* + *y* + *y* + *y* = 2*x* + 3*y*.
También tenemos que AB = AF + FB, BC = BF + FC, y AC = AF + FC = 2*x*. Luego x=AF=FC=y. Entonces AE = ED = DC = y = x.
Luego perímetro pentágono = 2x+3x = 5x.
Razón entre el perímetro del pentágono y el perímetro del triángulo ABC = 5*x* / (3*x*) = 5/3.
