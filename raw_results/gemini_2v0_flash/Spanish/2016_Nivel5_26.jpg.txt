Razonamiento: Sea A el área del triángulo, y a, b, c las longitudes de los lados correspondientes a las alturas h_a = 10, h_b = 11, h_c. Entonces A = (1/2) a * h_a = (1/2) b * h_b = (1/2) c * h_c.  De esto se deduce que 10a = 11b = h_c * c, y a = A/5, b = A/(11/2), c = A/(h_c/2) = 2A/h_c.  Como las longitudes de los lados de un triángulo deben satisfacer la desigualdad triangular, a < b + c, b < a + c, c < a + b.
a = A/5, b = 2A/11, c = 2A/h_c, tenemos que A/5 < 2A/11 + 2A/h_c, 2A/11 < A/5 + 2A/h_c, y 2A/h_c < A/5 + 2A/11.
Eliminando A, tenemos 1/5 < 2/11 + 2/h_c, 2/11 < 1/5 + 2/h_c, y 2/h_c < 1/5 + 2/11.
La última desigualdad implica 2/h_c < 11/55 + 10/55 = 21/55, o h_c > 2(55)/21 = 110/21 ≈ 5.238. Así que h_c > 5.238
La primera desigualdad implica 1/5 - 2/11 < 2/h_c, (11-10)/55 < 2/h_c, 1/55 < 2/h_c, h_c < 110.
La segunda desigualdad implica 2/11 - 1/5 < 2/h_c, (10-11)/55 < 2/h_c, -1/55 < 2/h_c, lo cual es siempre cierto.
