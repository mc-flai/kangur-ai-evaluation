Razonamiento:
El área del cuadrado es 2019^2. El área de cada baldosa es 673 * 1 = 673.
Necesitamos cubrir el área del cuadrado con las baldosas, entonces dividimos el área del cuadrado entre el área de cada baldosa: 2019^2 / 673.
Dado que 2019 = 3 * 673, entonces 2019^2 / 673 = (3 * 673)^2 / 673 = (9 * 673^2) / 673 = 9 * 673 = 6057.
