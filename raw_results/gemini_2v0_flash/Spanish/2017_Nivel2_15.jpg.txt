Razonamiento:
El área de un triángulo es (base * altura) / 2. El área original es (50 * 40) / 2 = 1000 cm². Debemos calcular el área resultante para cada opción y seleccionar la que dé el área más grande.

A) base = 50 + 20 = 70 cm, altura = 40 cm. Área = (70 * 40) / 2 = 1400 cm²
B) base = 50 cm, altura = 40 + 20 = 60 cm. Área = (50 * 60) / 2 = 1500 cm²
C) base = 50 + 10 = 60 cm, altura = 40 + 10 = 50 cm. Área = (60 * 50) / 2 = 1500 cm²
D) base = 50 + 15 = 65 cm, altura = 40 + 5 = 45 cm. Área = (65 * 45) / 2 = 1462.5 cm²
E) base = 50 + 5 = 55 cm, altura = 40 + 15 = 55 cm. Área = (55 * 55) / 2 = 1512.5 cm²

La opción E resulta en el área más grande.

Respuesta: E)
