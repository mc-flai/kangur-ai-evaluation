Raonament: Siguin P l'edat actual de Pere i J l'edat actual de Júlia. Se'ns diu que P + J = 91. També se'ns diu que J és el doble de l'edat que Pere tenia quan Júlia tenia l'edat que Pere té ara. Sigui x el nombre d'anys que han passat des d'aquell moment. Llavors, P - x és l'edat que tenia Pere quan Júlia tenia l'edat que té Pere ara, que és J - x. Per tant, J - x = P. A més, sabem que J = 2(P - x).

Tenim les equacions:
1) P + J = 91
2) J - x = P
3) J = 2(P - x)

De (2), x = J - P. Substituint això a (3), tenim J = 2(P - (J - P)) = 2(2P - J).
Per tant, J = 4P - 2J, o 3J = 4P. D'aquí, P = (3/4)J.

Substituint això a (1), tenim (3/4)J + J = 91, o (7/4)J = 91.
Llavors J = (4/7) * 91 = 4 * 13 = 52.

Resposta: D)
