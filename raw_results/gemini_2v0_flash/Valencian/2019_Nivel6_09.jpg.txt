Raonament:
Primer, desenvolupem l'expressió (a ◇ b) ◇ c utilitzant la definició de l'operació ◇:
(a ◇ b) ◇ c = (b - a) ◇ c = c - (b - a) = c - b + a

Després, desenvolupem l'expressió a ◇ (b ◇ c):
a ◇ (b ◇ c) = a ◇ (c - b) = (c - b) - a = c - b - a

Ara, igualem les dues expressions, ja que ens diuen que (a ◇ b) ◇ c = a ◇ (b ◇ c):
c - b + a = c - b - a
c - b + a - c + b + a = 0
2a = 0
a = 0

Per tant, la relació que és necessàriament certa és que a = 0.
