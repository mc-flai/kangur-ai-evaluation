a - 2 = x => a = x + 2
b + 3 = x => b = x - 3
c - 4 = x => c = x + 4
d + 5 = x => d = x - 5
e - 6 = x => e = x + 6

Ara comparem els valors de a, b, c, d, e:
a = x + 2
b = x - 3
c = x + 4
d = x - 5
e = x + 6

Com que tots els valors depenen de x, podem observar que el valor més gran serà el que tingui la major quantitat sumada a x. En aquest cas, e té x + 6, que és el valor més gran.
