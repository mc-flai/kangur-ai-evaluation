Raonament: L'equació |x - 2019| + |x - 2020| = 3 representa la suma de les distàncies de x als punts 2019 i 2020 és igual a 3. Si x està entre 2019 i 2020, llavors |x - 2019| + |x - 2020| = x - 2019 + 2020 - x = 1, que no és igual a 3. Per tant, x ha de ser més petit que 2019 o més gran que 2020.

Si x < 2019, llavors |x - 2019| + |x - 2020| = 2019 - x + 2020 - x = 4039 - 2x = 3. Així, 2x = 4036, i x = 2018.
Si x > 2020, llavors |x - 2019| + |x - 2020| = x - 2019 + x - 2020 = 2x - 4039 = 3. Així, 2x = 4042, i x = 2021.

Les dues solucions són x = 2018 i x = 2021. La suma d'aquestes solucions és 2018 + 2021 = 4039.
