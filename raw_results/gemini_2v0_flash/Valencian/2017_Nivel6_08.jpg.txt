Raonament:
Primer, expressem tan x i cot x en termes de sin x i cos x:
tan x = sin x / cos x
cot x = cos x / sin x
Llavors, tenim l'equació:
sin x / cos x + cos x / sin x = π
Combinem les fraccions per tenir un denominador comú:
(sin^2 x + cos^2 x) / (sin x cos x) = π
Sabem que sin^2 x + cos^2 x = 1, així que:
1 / (sin x cos x) = π
Per tant, sin x cos x = 1 / π
