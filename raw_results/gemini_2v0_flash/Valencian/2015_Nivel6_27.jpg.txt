Raonament: Observo que la pregunta presenta una suma i resta de quadrats consecutius. La idea és agrupar els termes per parelles:
(2015^2 - 2014^2) + (2013^2 - 2012^2) + ... + (3^2 - 2^2) + 1^2.

Recordem la identitat a^2 - b^2 = (a + b)(a - b). Aplicant-la a cada parella:
(2015 + 2014)(2015 - 2014) + (2013 + 2012)(2013 - 2012) + ... + (3 + 2)(3 - 2) + 1
= (2015 + 2014)(1) + (2013 + 2012)(1) + ... + (3 + 2)(1) + 1
= 2015 + 2014 + 2013 + 2012 + ... + 3 + 2 + 1.

Aquesta és la suma dels primers 2015 nombres naturals. La fórmula per a la suma dels primers n nombres naturals és n(n+1)/2.
En aquest cas, n = 2015. Per tant, la suma és 2015(2015 + 1)/2 = 2015 * 2016 / 2 = 2015 * 1008 = 2031120.
Aquest resultat coincideix amb l'opció D.

Resposta: D)
