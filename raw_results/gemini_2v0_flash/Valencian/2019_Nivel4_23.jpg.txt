Raonament: Traduïm l'enunciat a una equació:
3a + 4b = 10a + b
3b = 7a
b = (7/3)a

Com que a i b són xifres d'un número, han de ser enters entre 0 i 9.
Si a = 1, b = 7/3 (no enter)
Si a = 2, b = 14/3 (no enter)
Si a = 3, b = 7
Si a = 4, b = 28/3 (no enter)
Si a = 5, b = 35/3 (no enter)
Si a = 6, b = 14 (no pot ser > 9)

Per tant, a = 3 i b = 7.
a + b = 3 + 7 = 10
