*   A) Llima (80g) + pruna (50g) + raïm (410g) + plàtan (220g) = 760g
*   B) Pera (320g) + llima (80g) + raïm (410g) + plàtan (220g) = 1030g
*   C) Pruna (50g) + raïm (410g) + plàtan (220g) + pera (320g) = 1000g
*   D) Pera (320g) + plàtan (220g) + pruna (50g) + raïm (410g) = 1000g
*   E) Pera (320g) + llima (80g) + pruna (50g) + raïm (410g) = 800g
Tant l'opció C com la D sumen 1000g, però en l'opció C tenim 1000 grams mentre que en la D, també 1000.

Resposta: C)
