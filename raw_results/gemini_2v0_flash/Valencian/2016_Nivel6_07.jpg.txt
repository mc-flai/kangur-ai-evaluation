Raonament: Siguin $\angle CAB = \alpha$, $\angle ABC = \alpha$ ja que $CA = CB$. Siguin $\angle ADC = \beta$, $\angle BDC = \gamma$.
Com que $AD = AC$, el triangle $ADC$ és isòsceles, per tant, $\angle ACD = \angle ADC = \beta$.
En el triangle $ADC$, tenim $\angle CAD + \angle ACD + \angle ADC = 180^{\circ}$, per tant, $\alpha + \beta + \beta = 180^{\circ}$, és a dir, $\alpha + 2\beta = 180^{\circ}$.
Com que $DB = DC$, el triangle $BDC$ és isòsceles, per tant, $\angle DBC = \angle DCB = \alpha$.
En el triangle $BDC$, tenim $\angle DBC + \angle DCB + \angle BDC = 180^{\circ}$, per tant, $\alpha + \alpha + \gamma = 180^{\circ}$, és a dir, $2\alpha + \gamma = 180^{\circ}$.
Com que $D$ està sobre $AB$, tenim $\angle ADC + \angle BDC = 180^{\circ}$, per tant, $\beta + \gamma = 180^{\circ}$.
Ara tenim: $\alpha + 2\beta = 180^{\circ}$, $2\alpha + \gamma = 180^{\circ}$, $\beta + \gamma = 180^{\circ}$.
De la segona equació tenim $\gamma = 180^{\circ} - 2\alpha$. Substituint a la tercera equació, tenim $\beta + 180^{\circ} - 2\alpha = 180^{\circ}$, per tant, $\beta = 2\alpha$.
Substituint $\beta = 2\alpha$ a la primera equació, tenim $\alpha + 2(2\alpha) = 180^{\circ}$, per tant, $\alpha + 4\alpha = 180^{\circ}$, és a dir, $5\alpha = 180^{\circ}$, per tant, $\alpha = 36^{\circ}$.
L'angle que busquem és $\angle ACB = \angle ACD + \angle DCB = \beta + \alpha = 2\alpha + \alpha = 3\alpha = 3(36^{\circ}) = 108^{\circ}$.

Resposta: D)
