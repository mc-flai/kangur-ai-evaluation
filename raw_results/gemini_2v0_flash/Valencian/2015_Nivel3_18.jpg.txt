Raonament: Ara és l'any 2015. Hem de trobar l'any en què el producte de les xifres és 2 i calcular quants anys han passat des de 2015.

Provaré amb cada opció:
A) 2015 + 5 = 2020. Producte de les xifres: 2 * 0 * 2 * 0 = 0.
B) 2015 + 67 = 2082. Producte de les xifres: 2 * 0 * 8 * 2 = 0.
C) 2015 + 85 = 2100. Producte de les xifres: 2 * 1 * 0 * 0 = 0.
D) 2015 + 86 = 2101. Producte de les xifres: 2 * 1 * 0 * 1 = 0.
E) 2015 + 96 = 2111. Producte de les xifres: 2 * 1 * 1 * 1 = 2.

Resposta: E)
