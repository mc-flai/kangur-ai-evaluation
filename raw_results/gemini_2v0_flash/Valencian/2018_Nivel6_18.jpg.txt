Raonament: Com que a i b són les solucions de l'equació x² + x - 7 = 0, tenim a² + a - 7 = 0 i b² + b - 7 = 0. Per tant, a² = 7 - a i b² = 7 - b.
L'expressió que volem calcular és 3a² + 4b² + 2a + 3b + 1. Substituint a² i b² per les expressions equivalents obtenim:
3(7 - a) + 4(7 - b) + 2a + 3b + 1 = 21 - 3a + 28 - 4b + 2a + 3b + 1 = 50 - a - b.
Com que a i b són les solucions de x² + x - 7 = 0, sabem que la suma de les arrels a + b és igual a -1 (el coeficient de x canviat de signe).
Per tant, -a - b = -(a + b) = -(-1) = 1.
Així, l'expressió original esdevé 50 - a - b = 50 + 1 = 51.
