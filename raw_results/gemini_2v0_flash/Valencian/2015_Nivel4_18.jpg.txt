Sabem que $a < b < c < d < e < f$.
Ens donen $AD = CF$, que significa $d - a = f - c$.
També ens donen $BD = DF$, que significa $d - b = f - d$.
Volem veure quina de les opcions és conseqüència d'aquests fets.
A) $AB = BC$ equival a $b - a = c - b$, que és $2b = a + c$.
B) $BC = DE$ equival a $c - b = e - d$.
C) $BD = EF$ equival a $d - b = f - e$.
D) $AB = CD$ equival a $b - a = d - c$.
E) $CD = EF$ equival a $d - c = f - e$.

De $d - a = f - c$, tenim $d + c = f + a$.
De $d - b = f - d$, tenim $2d = f + b$.
Llavors $f = 2d - b$.
Substituint a la primera equació, $d + c = 2d - b + a$, d'on $c = d - b + a$.
Així, $c - b = d - 2b + a$ i $b - a = d - c$.
Ara analitzem l'opció B): $c - b = e - d$. Això implica $d - 2b + a = e - d$, d'on $2d - 2b + a = e$. No tenim prou informació per a saber si això és cert.

Per a analitzar l'opció E), usem que $f = 2d - b$ i $c = d - b + a$.
Llavors $d - c = d - (d - b + a) = b - a$.
I $f - e = 2d - b - e$. Per tant, la condició seria $b - a = 2d - b - e$, d'on $2b - a = 2d - e$.

Analitzem l'opció A): Volem veure si $b - a = c - b$, és a dir, si $2b = a + c$.
