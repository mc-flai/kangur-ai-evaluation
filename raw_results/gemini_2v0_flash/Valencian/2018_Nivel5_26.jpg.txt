Raonament:
Com que a i b són les solucions de l'equació x^2 + x - 7 = 0, tenim que a^2 + a - 7 = 0 i b^2 + b - 7 = 0.
Aleshores, a^2 = 7 - a i b^2 = 7 - b.
Substituint a l'expressió que volem trobar:
3a^2 + 4b^2 + 2a + 3b + 1 = 3(7 - a) + 4(7 - b) + 2a + 3b + 1 = 21 - 3a + 28 - 4b + 2a + 3b + 1 = 50 - a - b
Per les relacions de Viète, sabem que la suma de les arrels d'una equació quadràtica x^2 + x - 7 = 0 és -1. Així, a + b = -1.
Per tant, 50 - a - b = 50 - (a + b) = 50 - (-1) = 50 + 1 = 51.
