Raonament:
Sigui `L` el costat de l'hexàgon original. El seu perímetre és `6L`.
Sigui `r` el radi dels cercles. Llavors `L = 2r + 2r / sqrt(3) = 2r(1 + 1 / sqrt(3))`.  El costat del hexàgon interior és `2r`.  El perímetre del hexàgon interior és `6 * 2r = 12r`.

La relació entre els perímetres és `(6L) / (12r) = L / (2r) = (2r(1 + 1 / sqrt(3))) / (2r) = 1 + 1 / sqrt(3) = (sqrt(3) + 1) / sqrt(3) = (3 + sqrt(3)) / 3 = (3 + sqrt(3)) / 3`

Si observem la geometria de l'hexàgon, l'hexàgon interior té com a costat la distància entre els centres dels cercles. Si `L` és el costat de l'hexàgon gran, el costat de l'hexàgon petit serà `L * cos(π/6) / cos(π/6) = (L * sqrt(3)/2) / cos(π/6)`. La distància del centre a una aresta de l'hexàgon gran és `L * sqrt(3) / 2`. El radi dels cercles és `r = L * sqrt(3) / (2 * 3) = L * sqrt(3) / 6`. Per tant, la distància entre dos centres és `2 * L * sqrt(3) / 6 = L * sqrt(3) / 3`.
La relació entre els perímetres és `6L / (6 * L * sqrt(3) / 3) = 6L / (2L * sqrt(3)) = 3 / sqrt(3)`.
