Raonament: Siguin els cinc números a, b, c, d, i e.
Tenim les següents equacions:
a + b + c = 100
b + c + d = 200
c + d + e = 300
Volem trobar el valor de c, que és el nombre al centre de la llista.
Restem la primera equació de la segona:
(b + c + d) - (a + b + c) = 200 - 100
d - a = 100
Restem la segona equació de la tercera:
(c + d + e) - (b + c + d) = 300 - 200
e - b = 100

Ara sumem les tres equacions:
(a + b + c) + (b + c + d) + (c + d + e) = 100 + 200 + 300
a + 2b + 3c + 2d + e = 600
També tenim d - a = 100 i e - b = 100, de manera que d = a + 100 i e = b + 100.
Substituïm a la equació combinada:
a + 2b + 3c + 2(a+100) + (b+100) = 600
a + 2b + 3c + 2a + 200 + b + 100 = 600
3a + 3b + 3c = 300
3(a+b+c) = 300
a+b+c = 100 (que ja sabem)
Usem les equacions originals:
a + b + c = 100
b + c + d = 200
c + d + e = 300
Sumem les tres equacions:
a + 2b + 3c + 2d + e = 600
Sumem les tres equacions amb la resta de la primera a la segona i la segona a la tercera.
Aïllem c de la segona equació:
c = 200 - b - d
També: a = d - 100 i e = b + 100
Substituïm:
(d-100) + b + c = 100
b + c + d = 200
c + d + (b+100) = 300
c + d + b = 200
c = 200 - d - b
Si fem b + c + d = 200 i multipliquem per 2, 2b + 2c + 2d = 400
i a + 2b + 3c + 2d + e = 600
Subtraiem:
(a + 2b + 3c + 2d + e) - (2b + 2c + 2d) = 600 - 400
a + c + e = 200
substituïm e = b + 100 i a = d - 100
d - 100 + c + b + 100 = 200
d + b + c = 200
c = 200 - d - b

Ara sabem que c = 200 - b - d, i també que b + c + d = 200.
Per tant, la resposta és 100.
