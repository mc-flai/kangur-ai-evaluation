Aplicant la fórmula de l'àrea d'un triangle en funció de les àrees dels seus costats, es pot trobar la relació entre l'àrea del triangle APQR i el triangle ABC. Area(APQR) = |AP/AB * AQ/AD * AR/AC| Area(ABC). En aquest cas, el signe canvia en funció del nombre de reflexions entre els vectors AP, AB, AQ, AD, AR i AC. Tenim AP = 2AB, AQ = 3AD i AR = 4AC. La fórmula vectorial de l'àrea del triangle ABC té el valor de 0.5 * |vector AB x vector AC|. En aquest cas, el signe no importa donat que estem calculant un mòdul. La condició de D com a punt mig de BC implica que 2AD = AB + AC. Per tant, vector AD = (vector AB + vector AC) / 2, i vector AQ = 3 vector AD = (3/2)(vector AB + vector AC). Per tant, l'àrea del triangle APQR = 0.5 * |vector AP x vector AQ + vector AQ x vector AR + vector AR x vector AP| = |vector AP x vector AQ + vector AQ x vector AR + vector AR x vector AP|.  Vector AP x vector AQ = 2vector AB x (3/2 * vector AB + 3/2 * vector AC) = 3 * vector AB x vector AC. Vector AQ x vector AR = 3/2 * (vector AB + vector AC) x 4 * vector AC = 6 * vector AB x vector AC. Vector AR x vector AP = 4vector AC x 2vector AB = -8 vector AB x vector AC.  L'àrea del triangle APQR = |3 + 6 - 8| * Area(ABC) = |1| * Area(ABC). No obstant això, aquest resultat no és correcte. Per l'àrea de APQR = 0.5*|AP x AQ + AQ x AR + AR x AP|.
