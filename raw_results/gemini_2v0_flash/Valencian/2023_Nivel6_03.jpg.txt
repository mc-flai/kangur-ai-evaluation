Raonament: L'inequació donada és x³ < 64 < x².
Primer, analitzem x³ < 64. Prenent l'arrel cúbica a ambdós costats, obtenim x < ∛64, que significa x < 4.
Després, analitzem 64 < x². Prenent l'arrel quadrada a ambdós costats, obtenim √64 < |x|, que significa 8 < |x|. Això implica que x > 8 o x < -8.
Combinant les dues desigualtats, tenim que x < 4 i (x > 8 o x < -8). Per tant, tenim x < -8, ja que x no pot ser més gran que 8 a causa de la condició x < 4.
