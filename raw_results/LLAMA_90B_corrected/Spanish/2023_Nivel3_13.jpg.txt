Razonamiento: En el caso de un cubo, hay 6 caras, cada una de las cuales corresponde al área convexa de una cuña o pirámide. Para un solo cubo, pretende encontrar el número de color para las caras de tal forma que ningunas caras adyacentes (las caras que tienen un borde en común) tengan el mismo color. En el caso de las caras del dado, debemos considerar el problema de los colores de las caras con una restricción. Supongamos que se le entregan tres colores distintos, representados en este caso por los números: 1, 2 y 3, en este orden. 1. Selecciona cualquier cara de la superficie del dado. 2. Colorea la cara seleccionada con el color número 1. 3. De las cuatro caras restantes, selecciona otra adyacente a la cara coloreada previamente (con borde en común) y colórala con el color 2. 4. Tomando ahora como referencia la cara coloreada con el color 2... Para otras tres caras adyacentes a la cara número 1, asigna cada cara con el color 3. 5. Ahora ya solo te queda una cara que con el número consecutivo es coloreada con el color número 2. 6. Así, todas las caras adyacentes son coloreadas con colores distintos. Como vemos, se puede colorear el dado de tal forma que ninguna cara que es adyacente tenga el mismo color, utilizando solo 3 colores distintos.
